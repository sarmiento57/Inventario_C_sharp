# Inventario Dashboard

Aplicación de escritorio en **C# WinForms** con **Guna UI2** para la gestión de inventario.  
Permite administrar productos, categorías y movimientos con una interfaz moderna y fácil de usar.

---

## Características
- CRUD de productos, categorías y movimientos.
- Validación de campos obligatorios y numéricos.
- Interfaz con Guna UI2.
- Conexión a SQL Server.
